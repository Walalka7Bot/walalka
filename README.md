# walalka ForexPro Bot

A Telegram trading bot that sends PRO signals for:
- 5 Forex pairs (EURUSD, GBPUSD, etc.)
- BTC, ETH, XAU, XAG, XRP

## Features
- TP, SL, EN levels
- Chart preview via quickchart
- Lot size calculator
- Voice alerts
- Confirm/Ignore button interactions

## Usage
Run locally:
```bash
python main.py
